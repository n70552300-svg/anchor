# Anchor Trust Rules

Anchor is built on privacy, security, and transparency.

## Anchor never
- Stores or asks for passwords
- Sells or shares user data
- Acts on accounts without user approval
- Hides what it accessed or why it made a recommendation

## Anchor defaults to
- Read-only access
- Minimal data collection
- Clear explanations (“why this was flagged”)
- Easy revocation of access

## Operator control (permission dial)
- Observe: discovery + risk only
- Hub: open apps/sites + guidance
- Connect: user links supported services
- Act: only with explicit confirmation per action
